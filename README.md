# AIdent2
# 🤖 AI Text Guard  **AI Text Guard** — это инструмент для определения авторства текста. Он помогает понять, был ли текст написан человеком или сгенерирован нейросетью (GPT-4, Claude, Llama).  ### 🚀 Функции * **Deep Analysis:** Проверка текста на вероятность генерации. * **Visual Metrics:** Наглядные графики сложности и вариативности. * *
